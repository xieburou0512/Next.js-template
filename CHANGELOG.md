# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Project documentation improvements
- MIT License
- .nvmrc for Node version management
- Enhanced package.json metadata

## [0.1.0] - 2025-01-07

### Added
- Next.js 16.0.7 with App Router
- Bun runtime support with Turbopack
- TypeScript 5.9.3 for type safety
- Internationalization with LinguiJS (English, Simplified Chinese)
- Tailwind CSS 3.4.18 for styling
- PWA support with Serwist
- State management with Zustand
- Data fetching with SWR
- HTTP client with ky
- Framer Motion for animations
- React Hooks library (ahooks)
- ESLint and Prettier for code quality
- Husky and lint-staged for git hooks
- Multi-environment configuration (dev, test, production)
- Responsive design with mobile/desktop detection
- Service Worker for offline support
- Bundle analyzer for performance optimization

### Features
- Dynamic language routing with middleware
- Server-side and client-side i18n support
- Custom theme system (light/dark mode)
- Optimized build configuration
- Cookie-based preference persistence
- Type-safe API client with error handling
- Modular project structure
- Environment-specific builds
